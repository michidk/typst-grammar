# Typst Grammar


> [!WARNING]  
> This grammar in this repository is very outdated and does not reflect the current state of the Typst language. Syntax highlighting issues are to be expected. 

> [!IMPORTANT]  
> Maintainers wanted! If you're interested in taking over, please get in touch.


This repository contains the Typst grammar definition for the [Typst](https://typst.app/) typesetting language, used for syntax highlighting on the GitHub website through [GitHub Linguist](https://github.com/github/linguist).

Original version taken from https://github.com/nvarner/typst-lsp (heavily modified).
